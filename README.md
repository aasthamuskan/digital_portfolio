# Sunny Tyagi - Portfolio Website

A "Next-Generation" interactive portfolio website built with **Next.js 15**, **Tailwind CSS**, **Framer Motion**, and **GSAP**.

## 🚀 Features

-   **Kinetic Typography Hero**: Mouse-tracking parallax and text reveals.
-   **Realtime Coding Dashboard**: Live stats from **GitHub** (simulated contribution graph) and **LeetCode** (submission heatmap).
-   **3D Elements**: Floating skill nodes and tilt-effect project cards.
-   **Performance Optimized**: Uses `Lenis` for smooth scrolling and optimized animations.
-   **Terminal Logs**: specialized "hacker-style" logs viewer.

## 🛠️ Tech Stack

-   **Framework**: Next.js 15 (App Router)
-   **Styling**: Tailwind CSS
-   **Animation**: Framer Motion, GSAP
-   **Icons**: Lucide React

## 📦 Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```
3.  Open [http://localhost:3000](http://localhost:3000).
